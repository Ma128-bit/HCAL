# MPGD-HCAL signal fit
This repository contains the files necessary to perform the fit on the signals on each strip/pad with Condor.

## Setting the environment

```
cmsrel CMSSW_13_2_4
cd CMSSW_13_2_4/src
cmsenv
git clone https://github.com/Ma128-bit/HCAL.git .
mkdir dictionaries
mkdir Root_File
```
